🎓 North State University Website

A modern, fully responsive, multi-page university website designed to showcase academic programs, campus locations, facilities, student feedback, and an integrated blog/contact platform.
This project was built as a clean frontend implementation using HTML5, CSS3, and lightweight JavaScript to provide smooth navigation and responsive design.


📸 Pages Overview

The website includes several structured pages designed to represent a complete university portal.

Home
—  Overview of the university
—  Featured courses and facilities
—  Student testimonials
About Us
—  University mission and vision
—  Institutional goals and core values
Courses
—  Undergraduate programs
—  Postgraduate programs
—  International study paths
Blog
—  Certificate and online program articles
—  Category sidebar navigation
—  Interactive comment form layout
Contact Us
—  Embedded map showing campus location
—  Contact form for user inquiries
  
✨ Features
—  Fully responsive design that works on desktop, tablet, and mobile devices
—  Pure CSS Flexbox layouts without external frameworks
—  Animated mobile navigation menu controlled with JavaScript
—  Modern hover effects and card shadow transitions
—  Image overlay effects for campus gallery sections
—  Clean and consistent typography using Google Fonts
  
🛠 Technologies Used
 — HTML5 – semantic page structure
 — CSS3 – layouts, styling, media queries, and transitions
 — JavaScript (ES6) – mobile menu toggle functionality
 — Font Awesome 4.7 – scalable vector icons
 — Google Fonts – Poppins and Oswald
